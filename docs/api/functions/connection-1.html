函数 connection

数据库连接配置

创建一个 PostgreSQL 数据库连接实例:

  • 使用环境变量 DATABASE_URL 作为连接字符串
  • 当进行数据库迁移或数据填充时,将最大连接数限制为1
  • 在数据填充过程中忽略数据库通知消息
