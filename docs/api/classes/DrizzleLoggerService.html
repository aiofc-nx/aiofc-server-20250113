私有interpolateQuery

  • interpolateQuery(query: string, parameters?: any[]): string

  • 查询参数插值处理

    主要机制:

    1. 参数替换
    • 接收原始SQL查询和参数数组
    • 遍历参数数组进行替换处理
    • 使用正则表达式匹配$1,$2等占位符
    1. 参数格式化
    • 对象类型参数转换为JSON字符串
    • 非对象类型参数直接使用值
    • 所有参数值添加单引号包裹
    1. 安全处理
    • 检查参数数组是否存在且有值
    • 保持原始查询不变如果没有参数

    参数

    • query: string

      原始SQL查询字符串

    • 可选parameters: any[]

      查询参数数组

    返回 string

    插值后的完整SQL查询
