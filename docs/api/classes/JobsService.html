类 JobsService

Jobs服务层

主要机制:

  1. 依赖注入
  • 使用@Injectable()装饰器标记为可注入的服务
  • 通过构造函数注入JobDao数据访问对象
  1. 数据库操作封装
  • 所有方法都是异步的,返回Promise
  • 通过JobDao封装具体的数据库操作
  • 使用TypeScript类型确保类型安全
  1. CRUD操作实现
  • 实现标准的增删改查功能
  • 支持单条和批量操作

要点:

  • 业务逻辑与数据访问分离，我们可以通过注入不同的Dao来实现不同的数据库操作（甚至实现使用不同的orm和数据库）
  • 统一的错误处理
  • 类型安全的数据传输
