类 TenantConnectionPool

租户连接池类

机制说明：

  1. 使用静态Map存储所有租户的数据库连接
  2. 每个租户都有独立的连接池和schema
  3. 采用懒加载方式，首次访问时才创建连接
  4. 通过search_path参数实现多租户数据隔离

  • 关闭所有租户的数据库连接
