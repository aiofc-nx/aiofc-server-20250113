类 JobResponseDto

基础数据传输对象(DTO)抽象类 定义这个抽象类是为了统一DTO的验证和清理方法

机制说明:

  1. 使用Zod进行数据验证和类型推断
  2. 提供统一的数据验证和清理方法
  3. 所有DTO类都继承自此基类

主要功能:

  • 定义统一的schema验证规则
  • 提供完整和部分数据的验证方法
  • 清理多余的数据字段
