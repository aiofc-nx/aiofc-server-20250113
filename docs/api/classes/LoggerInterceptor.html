类 LoggerInterceptor

自定义日志拦截器

主要机制:

  1. 请求拦截
  • 使用@Injectable()装饰器标记为可注入服务
  • 实现NestInterceptor接口进行请求拦截
  • 通过ExecutionContext获取请求和响应对象
  1. 日志记录
  • 使用NestJS内置Logger进行日志输出
  • 记录请求接收和响应完成的日志
  • 通过LoggerUtils工具类格式化日志消息
  1. 性能监控
  • 使用ClsService存储请求开始时间
  • 计算请求处理耗时
  • 在响应时输出耗时信息

要点:

  • 统一的日志格式和处理
  • 请求全生命周期的监控
  • 错误情况下也保证日志输出
  • 使用响应式编程处理异步流程
