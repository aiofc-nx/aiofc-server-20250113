类 DrizzlePostgresModule

DrizzlePostgres模块

机制说明:

  1. 通过@Global()装饰器将模块声明为全局模块
  2. 继承ConfigurableModuleClass以支持动态模块配置
  3. 提供同步(register)和异步(registerAsync)两种注册方式

主要功能:

  • 注册DrizzlePostgresService服务
  • 根据配置创建并导出Drizzle实例
  • 支持多实例通过tag区分
